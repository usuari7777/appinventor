Chateame

Objetivos

  • Crear una app que permita mandar mensajes entre dos dispositivos.

 Nosotros crearemos una  aplicación muy sencilla, si lo que pretendes es crear un Whatsapp o un Telegram haz uso de tus avanzados conocimientos de programación en  Android y ¡ánimo!

 ¡CUIDADO!: en esta aplicación usaremos el componente "Texting" de AppInventor. Éste componente no dispone de un servidor donde almacenar los mensajes, por lo que realmente lo que estamos haciendo es mandar SMS. De hecho mandamos un SMS cada vez que invocamos el procedimiento "SendMessage" (para mandar un mensaje). Así que ... ¡cuidado con tu factura telefónica!

 

Actividad

Entra en AppInventor, selecciona el proyecto "contactos" de la sección anterior contactando. Ve al menú "Projects" --> "Save project as..." y guárdalo con el nombre "uasap".

No creas que estamos plagiando ningún nombre, "uasap" es un acrónimo inglés de la expresión: "u a so amazing, pal" (You are so amazing, pal).

Borra el "TableArrangement1" que tenía sentido para la aplicación "contactos" y sustitúyelo por:

  • Una etiqueta donde recibiremos el mensaje que nos han mandado. Llámala "LabelReceived" y estas propiedades:
    • FontSize: 25
    • Text: -----------
    • Width: Fill Parent
  • Una caja de texto donde escribiremos el mensaje de texto a enviar. Renómbrala "TextBoxSent" y propiedades:
    • FontSize:25
    • Hint: Escribe un mensaje 
    • MultiLine: True
    • Width: Fill Parent
  • Un botón llamado "ButtonSend" y con el texto "Enviar mensaje"
